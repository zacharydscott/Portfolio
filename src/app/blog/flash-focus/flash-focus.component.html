Flash Focus: Building a Full Stack CRUD Application

It turns out that all you really need to make a full CRUD application is JavaScript. NodeJS is so ubiquitous that if you do any UI work, you'll have your environment already set up. But Node is useful for more than serving your local React project, obviously. Express makes it easy to set up a server. So, if you're a front end developer, there's no reason not to dip your toes into the world server development as well.
